# Digital Support Systems, Inc. — Website

Official website for **Digital Support Systems, Inc.**, built with **Next.js** and **Tailwind CSS**.

This project serves as the foundation for the company’s public-facing site, highlighting:
- IP-based access control solutions (Allegion / ISONAS)
- IT infrastructure design and development
- Systems integration and professional services

---

## Tech Stack
- **Next.js** (App Router)
- **React**
- **Tailwind CSS**
- **TypeScript**
- **Node.js**
